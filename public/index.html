MAXFLOW API Documentation

This API Provides API endpoints for MAXFLOW Application

POST /user/register

Register a new user.

POST https://maxflow-server-production.up.railway.app/user/register

Request body should contain the following fields:

{
    "username": "string",
    "email": "string",
    "password": "string"
}

POST /user/login

Log in a user.

POST https://maxflow-server-production.up.railway.app/user/login

Request body should contain the following fields:

{
    "email": "string",
    "password": "string"
}

GET /task/get/:userId

Get all tasks for a user.

GET https://maxflow-server-production.up.railway.app/task/get/:userId
